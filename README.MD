# Night Vision+ (Safe for Achievements)

A lightweight Minecraft Bedrock Edition addon that provides permanent Night Vision while maintaining a clean gameplay experience.

> **Status:** In Development

## Features

- Permanent Night Vision
- Enabled by default
- Stick toggle
- Hidden potion particles
- Automatic reactivation after respawn
- Lightweight and optimized
- Achievement-friendly addon

## Requirements

- Minecraft Bedrock Edition 1.21.130 or later
- `@minecraft/server` Script API 2.3.0

## Installation

1. Download the latest release.
2. Import the `.mcpack` file into Minecraft.
3. Enable the addon in your world.
4. Create or load a world with the addon enabled.

## Usage

- Night Vision is enabled automatically when joining a world.
- Use a stick to toggle Night Vision on or off.
